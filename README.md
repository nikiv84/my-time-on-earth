# My Time on Earth
![No more Tabs](/assets/icon.png)

A React app for the exibition: Vuk Vidor - 20.000 Days on Earth

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install WizardApp.

```
$ yarn install
```

## Starting the app

```
$ yarn start
```
